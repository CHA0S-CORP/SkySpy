# SkySpy Bug Audit

Parallel multi-agent audit of backend (Django services, API/auth, Socket.IO, Celery), frontend (hooks, components), Go TUI client, and Python packages. ~90 real bugs. Each item has an exact `file:line`, severity, confidence, the concrete failure, and a fix hint.

**Recurring root cause:** several flagship features are wired up and unit-tested but **dead in production** â€” a missing call site, wrong key name, or missing import â€” and broad `except Exception` / empty `catch` handlers hide the failure. CI is green because tests call the code directly.

**How to use this file:** each `- [ ]` is an independent fix task. Check it off when patched. Run the test command noted in the repo `CLAUDE.md` for the touched area. Don't remove entries â€” mark them `- [x]` with a one-line note.

Severity: **critical** = feature entirely dead / all-garbage output Â· **high** = silent data loss, credential leak, lockout Â· **medium** = wrong results in common cases Â· **low** = edge cases, dead code, log noise.
